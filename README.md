# Мои задачи — PWA

Готовое устанавливаемое веб-приложение со списком задач.

## Возможности

- работает на iPhone, Android и компьютере;
- устанавливается на домашний экран;
- работает без интернета после первого открытия;
- сохраняет задачи в `localStorage`;
- адаптировано под безопасные зоны iPhone.

## Локальный запуск

Service Worker не работает при открытии файла напрямую через `file://`. Запустите локальный сервер.

### Python

```bash
cd pwa-task-app
python3 -m http.server 8080
```

Откройте:

```text
http://localhost:8080
```

### Node.js

```bash
npx serve .
```

## Установка на iPhone

1. Разместите папку на хостинге с HTTPS.
2. Откройте адрес в Safari.
3. Нажмите «Поделиться».
4. Выберите «На экран “Домой”».

## Публикация

Подойдут GitHub Pages, Netlify, Vercel, Cloudflare Pages или любой HTTPS-хостинг.

## Основные файлы

- `index.html` — разметка;
- `styles.css` — оформление;
- `app.js` — логика и локальное сохранение;
- `manifest.webmanifest` — параметры установки;
- `service-worker.js` — кеширование и офлайн-режим;
- `icons/` — иконки приложения.
