# Планировщик — PWA

Устанавливаемое веб-приложение с боковым меню и четырьмя независимыми разделами:

- Календарь;
- Регулярное;
- Планирование;
- Список задач и целей.

На текущем этапе каждый раздел использует одинаковый интерфейс списка задач, но данные разделов хранятся отдельно в `localStorage`.

## Возможности

- работает на iPhone, Android и компьютере;
- устанавливается на домашний экран;
- работает без интернета после первого открытия;
- сохраняет задачи отдельно для каждого раздела;
- переносит старые задачи в раздел «Список задач и целей»;
- адаптировано под безопасные зоны iPhone.

## Локальный запуск

Service Worker не работает при открытии файла напрямую через `file://`. Запустите локальный сервер.

### Windows / Python

```bat
cd путь\к\папке\Sheduler
py -m http.server 8080
```

Откройте `http://localhost:8080`.

## Публикация

Замените файлы репозитория новыми версиями и создайте коммит. GitHub Pages обновится автоматически после публикации изменений в ветку `main`.
